# <cg>Edit</c> <cf>Tools</c>

## About
Edit Tools adds several useful tools to the Geometry Dash editor

It includes tools for:

- connecting objects between two selected points
- filling an area with objects
- replacing one object type with another
- creating circles of objects
- connecting triggers to objects
- disconnecting triggers from objects

The mod also includes a optional toolbar buttons, configurable hotkeys, and settings for each tool

You can customize things like which object's properties are used, grid alignment, spacing, rotation, and other tool behavior
